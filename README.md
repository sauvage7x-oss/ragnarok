# Ragnarok Online Website

A beautiful, responsive website for Ragnarok Online with a modern design and full file structure.

## 📁 Project Structure

```
ragnarok/
├── index.html              # Home page
├── css/
│   ├── style.css          # Main styles
│   └── navbar.css         # Navigation styles
├── js/
│   └── script.js          # JavaScript functionality
├── pages/
│   ├── download.html      # Download page
│   ├── guide.html         # Guide page
│   ├── community.html     # Community page
│   └── contact.html       # Contact page
├── assets/
│   └── images/            # Image folder
└── README.md             # This file
```

## 🎨 Features

- ✨ Modern and responsive design
- 🎮 Ragnarok Online themed
- 📱 Mobile-friendly
- ⚡ Smooth animations and transitions
- 🔗 Easy navigation between pages
- 📧 Contact form
- 🎯 Clean file organization

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Enjoy!

## 📄 Pages

- **Home** - Welcome page with featured sections
- **Download** - Game download options
- **Guide** - Beginner's guide and tips
- **Community** - Community information
- **Contact** - Contact form

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## 📝 Notes

This is a frontend template. To make it fully functional, you may want to:
- Add actual game images to `assets/images/`
- Connect the contact form to a backend service
- Add more content and pages as needed
- Deploy to a hosting service
